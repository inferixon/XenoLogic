# Обов'язкове завдання 03

## Опис

-У цьому завданні велика увага приділяється плануванню та дизайну веб-сайту, який не обов’язково знаходиться в нашій зоні комфорту, темою завдання є космос, який є великою темою з багатьма можливостями для того, що могло б стати основним фокусом.
-Ми трохи розглянули альтернативні сторінки, які ми можемо створити, де їх можна розділити на дві з інформативним або фантазійним, що створює певну основу для того, куди ми можемо рухатися з вмістом для сторінки. Незважаючи на те, що вміст вибрано, ми можемо багато зробити індивідуально, щоб створити класний веб-сайт і використовувати методи, які ви хочете просувати вибрану тему в просторі.
Щоб забезпечити акуратний і зручний веб-сайт, ми використовуватимемо сучасні методи, такі як CSS Flex або CSS Grid (сітку буде переглянуто наступного тижня 31.03) для розміщення вмісту, Google Fonts (або еквівалент) для типографіки та медіа-запитів для адаптивного дизайну для комп’ютерів і мобільних пристроїв.


## Вимоги до подання для веб-сайту

-Одна сторінка: щонайменше 3 розділи з вмістом
-Багатосторінковий: щонайменше 3 сторінки вмісту

## Крок 1: Створіть базову структуру
-Використовуйте свій дизайн Figma як шаблон і створіть структуру HTML, яка відповідає макету та вмісту.
-Налаштуйте CSS для розташування, розмірів і базового стилю.
-Використовуйте Flexbox або Grid, щоб організувати макет, як ви планували.

## Крок 2: Зробіть веб-сайт адаптивним і унікальним


-Введіть медіа-запити, щоб веб-сайт працював як на мобільному пристрої, так і на комп’ютері.
-Використовуйте принаймні один зовнішній шрифт від Google Fonts або подібний.

## Крок 3: Зробіть дизайн більш інтерактивним
-Додайте ефекти наведення курсора на кнопки або зображення (наприклад, змініть колір, видимість або масштаб).
-Додайте кнопки, поля або розділи «Додаткова інформація», до яких вони ведуть (але не потребують фактичної функції).
-Бонусні завдання: (необов'язково)

## Якщо ви хочете випробувати себе та дізнатися більше, ви можете спробувати один або кілька з них

-Створіть змінні CSS для кольорів або розмірів шрифту, що полегшить підтримку дизайну.
— Додайте CSS-анімацію або перехід (наприклад, плавне наближення до елементів або ковзний перехід при наведенні).
-Використовуйте JavaScript, щоб зробити кнопку заклику до дії функціональною (наприклад, показати/приховати повідомлення).
-Створіть кнопку «прокручування вгору», яка з’являється, коли користувач прокручує сторінку вниз.
