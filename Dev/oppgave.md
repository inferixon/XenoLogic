# Oppgave 02 Brief

Goal: plan and design a simple website in Figma before implementation starts.

## Assignment Summary

- Start with a desktop wireframe.
- Upgrade that wireframe into a finished desktop design.
- Repeat the same process for mobile.
- Keep the layout realistic enough to build later with HTML and CSS.
- Use a space-related topic.

## Required Structure

- Header: logo + navigation
- Hero: title + image or video + short description
- At least two content sections or content pages
- Footer: contact info, links, and optional fictional social media

## Topic Notes

Suggested directions:

- Planets
- Stars
- Spacecraft
- Astronauts
- Space games or movies
- Any other space-related angle

## Delivery

- Submit the Figma share link.
- Use the provided Figma template sizes for the four frames.
