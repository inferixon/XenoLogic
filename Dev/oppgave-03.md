# Obligatorisk oppgave 03

## Beskrivelse

- I denne oppgaven så ligger mye av fokuset på å planlegge og strukutrere en nettside som ikke nødvendigvis er innen for komfortsonen våres, oppgavens tema er verdensrommet som er et stort tema med mange muligheter på hva som kan være hovedfokus.
- Vi gikk litt gjennom alternative sider vi kan lage, der det kan deles opp i to med informativ eller fantasi - som lager litt grunnmuren på hvor vi kan gå med innhold for siden. Selvom innholdet er valgt, så er det mye vi kan gjøre individuelt for å lage en kul nettside og bruke teknikker som dere har lyst for å fremme deres valgte tema innen for verdensrommet.

For å sikre en ryddig og brukervennlig nettside skal vi bruke moderne teknikker som CSS Flex eller CSS Grid (Grid gåes gjennom neste uke 31.03) for plassering av innhold, Google Fonts(eller tilsvarende) for typografi og media queries for responsivt design til desktop og til mobil.


## Innleverings krav for nettsiden

- Singlepage: Minimum 3 seksjoner med innhold
- Multipage: Minimum 3 sider med innhold

## Steg 1: Lag grunnstrukturen

- Bruk Figma-designet ditt som mal og lag HTML-strukturen som matcher layout og innhold.
- Sett opp CSS for plassering, størrelser og grunnleggende stil.
- Bruk Flexbox eller Grid for å organisere layouten slik du planla.

## Steg 2: Gjør nettsiden responsiv og unik


- Legg inn media queries slik at nettsiden fungerer både på mobil og desktop.
- Bruk minst én ekstern font fra Google Fonts eller lignende.

## Steg 3: Gjør designet mer interaktivt


- Legg til hover-effekter på knapper eller bilder (for eksempel endre farge, synlighet eller skalering).
- Legg til knapper, bokser eller "Mer informasjon"-seksjoner som  de leder videre (,men som trenger ikke faktisk funksjonalitet).
- Bonusutfordringer: (frivillig)

## Hvis du vil utfordre deg selv og lære mer, kan du prøve én eller flere av disse


- Lag CSS-variabler for farger eller fontstørrelser, slik at det blir lettere å vedlikeholde designet.
- Legg til CSS-animasjon eller overgang (f.eks. fade-in på elementer eller glidende overgang på hover).
- Bruk JavaScript til å gjøre en Call-To-Action-knapp funksjonell (f.eks. vis/skjul en melding).
- Lag en "scroll-to-top"-knapp som dukker opp når brukeren har scrollet nedover på siden.
